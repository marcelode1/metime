OneDrive Cloud Backup

Back up each project's photos and comments to a {{ onedrive_root_folder }} folder in OneDrive. Files go straight to the cloud (nothing is stored on a PC). Each project then has Create Backup Folder and Backup Data buttons, and shows when it was last backed up.

📖 First-time setup — how to connect OneDrive (read this)
  1. Register the app with Microsoft (one time). Go to portal.azure.com → Microsoft Entra ID → App registrations → New registration.
    • Name: ProjectONus
    • Supported account types: “Accounts in any organizational directory and personal Microsoft accounts”
    • Redirect URI → platform Web → paste exactly:
      {{ onedrive_redirect_uri }}
  2. Open the new app → Overview → copy the Application (client) ID.
  3. Certificates & secrets → New client secret → copy the secret’s Value (not the ID).
  4. Paste both into the boxes below and click Save OneDrive Settings. (Leave Tenant as common — it works for both personal and business OneDrive.)
  5. Click Connect OneDrive and sign in with the Microsoft account that owns the OneDrive. Approve access. The {{ onedrive_root_folder }} folder is created automatically.
  6. Go to any project → Create Backup Folder, then Backup Data. Re-pressing only sends new items — nothing is uploaded twice.

No files are kept on any PC — uploads go directly to OneDrive’s cloud. Business/work accounts may need a Microsoft 365 admin to “grant consent” once.
