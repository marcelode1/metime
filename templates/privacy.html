Privacy Policy

Effective Date: May 2026

ProjectONus (“we”, “our”, or “us”) provides project management, room documentation, communication, photo storage, audio notes, and collaboration tools for contractors, employees, and customers.

Information We Collect

ProjectONus may collect:

  • User account information
  • Project information
  • Customer information
  • Photos uploaded by users
  • Audio recordings uploaded by users
  • Comments and room notes
  • Device/browser information required for app functionality

How Information Is Used

Information is used only for:

  • Project management
  • Team collaboration
  • Customer communication
  • Field documentation
  • Internal reporting
  • System security and permissions

We do not sell personal information.

Storage

Files and project data may be stored using secure third-party cloud providers, including Render and Supabase.

Permissions

Project administrators may control:

  • User permissions
  • Access to comments
  • Access to photos
  • Access to audio
  • Project visibility

Mobile Features

The mobile app may request access to:

  • Camera
  • File storage
  • Microphone

These permissions are used only for app functionality.

Security

We take reasonable measures to protect user information and project data.
